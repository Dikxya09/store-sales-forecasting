# Sales Forecasting Dashboard

An end-to-end sales forecasting project featuring data analysis, visualization, and an interactive dashboard built using Streamlit and machine learning.

---

##Features

* Sales data analysis and visualization
* Monthly sales trend analysis
* Machine learning-based sales prediction
* Interactive dashboard using Streamlit
* Clean and user-friendly interface

---

##Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Streamlit

---

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/sales-forecasting-dashboard.git
```

2. Navigate to project folder:

```
cd sales-forecasting-dashboard
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the app:

```
streamlit run app.py
```
